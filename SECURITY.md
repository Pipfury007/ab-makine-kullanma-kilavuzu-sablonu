# Güvenlik Politikası

## Desteklenen kapsam
Bu repo dokümantasyon şablonları ve yardımcı script’ler içerir. Güvenlik bildirimlerini şu alanlar için kabul ederiz:
- /scripts altındaki script’ler (tipik kullanımda istismar riski varsa)
- GitHub Actions workflow’ları (güvenlik riski oluşturuyorsa)

## Zafiyet bildirimi
Hassas güvenlik problemleri için lütfen public issue açmayın.

Bunun yerine:
1) Minimum detayla “security” etiketli bir issue açın, veya
2) Maintainer ile GitHub profil kanalları üzerinden iletişime geçin.
